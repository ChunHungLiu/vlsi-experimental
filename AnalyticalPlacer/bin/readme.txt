This tool is aimed for solving Wirelength-driven Standard-Cell Placement Problem.
Benchmarks have to be given in a bookshelf format [http://vlsicad.ucsd.edu/GSRC/bookshelf/Slots/Placement/plFormats.html].

Now we have created a Win32 console applications which should be used in the following way:

Whole standard-cell placer:
itlDragon -f <benchmark_name>.aux
