Файл autoexp.dat содержит настройки отладчика для Visual Studio 2005.
В нем прописаны правила отображения различных типов в окнах отладки.

Для установки необходимо заменить файл
\Program Files\Microsoft Visual Studio 8\Common7\Packages\Debugger\autoexp.dat

Не забудьте сделать бэкап оригинального файла.


Install.bat делает установку файла автоматически.
